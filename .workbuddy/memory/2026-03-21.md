## PicLayout 图片排版工具 - 初始开发完成

### 项目概述
- 通用图片排版 Web 工具，基于 React + TypeScript + Vite + Fabric.js
- 深色主题，三栏布局（左侧模板、中间画布、右侧图片）
- 支持 12 种预设画布尺寸（含 7 寸照片）+ 自定义尺寸
- 10 个内置排版模板 + 自定义行列网格
- 支持导出 PNG/JPG/PDF，可设置 DPI 和质量

### 技术栈
- React 18 + TypeScript + Vite 5
- Fabric.js 6（Canvas 渲染引擎）
- Zustand（状态管理）
- Tailwind CSS 3.4.17
- jsPDF（PDF 导出）
- react-dropzone（拖拽上传）
- lucide-react（图标）

### 目录结构
- `src/types/` - 核心类型定义
- `src/stores/` - Zustand 状态（canvas/template/image/history）
- `src/components/` - UI 组件（CanvasPanel/Toolbar/TemplatePanel/ImagePanel/Dialogs）
- `src/hooks/` - useCanvas/useExport
- `src/templates/` - 内置模板配置
- `src/utils/` - 常量和工具函数
